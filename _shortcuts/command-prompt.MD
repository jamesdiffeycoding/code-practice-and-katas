### Command Prompt

| **Function**               | **Shortcut**     | **Notes** |
| -------------------------- | ---------------- | --------- |
| List files and directories | `dir`            |           |
| Open VSCode                | `code`           |           |
| Delete file                | `del`            |           |
| Change directory           | `cd`             |           |
| Rename file                | `ren`            |           |
| Clear screen               | `cls`            |           |
| Copy file                  | `copy`           |           |
| Move file                  | `move`           |           |
| Make folder                | `mkdir`          |           |
| Show text file contents    | `type`           |           |
| Get command help           | `commandName /?` |           |
